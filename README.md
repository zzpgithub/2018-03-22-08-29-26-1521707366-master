# Java编程基础 - 集合运算

## 练习描述
+ 根据测试目录下的测试代码，完成实现使得测试代码可以运行通过。
+ 不修改测试代码，只修改实现代码，使得测试全部通过。

## 环境要求
+ git
+ Java 8

## 如何开始

+ git clone 模板库地址后初次下载完需要安装依赖：

> ./gradlew build -x test

+ 运行测试：

> ./gradlew clean test  跑全部的测试
  完成一个函数的测试可以单独运行，推荐使用快捷键 Ctrl+shift+R (mac) 

注意：初次运行test 会发现测试全部失败
